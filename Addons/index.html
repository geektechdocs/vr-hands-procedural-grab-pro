    📦 VR74 Rifle Add-On

    Description Advanced weapon interaction add-on built specifically for VR Hands Procedural Grab Pro.

    This expansion extends the procedural hand system with a mechanically authentic KA-74 rifle interaction setup, fully integrated with physics-based grab logic and two-hand support.

    Designed for Unreal Engine 5 VR projects, this add-on provides a production-ready rifle interaction framework

    PLAYABLE DEMO

    FAB PAGE

    ⚠️ VR Hands Procedural Grab Pro Plugin is required for this framework!

    🔧 Core Features

    • Niagara Decal Spawning (Pooling System)
    • Digital Bullet Counter
    • Functional bolt pull and release system
    • Magazine extraction and insertion mechanics
    • Safety switch state logic
    • Trigger validation system
    • Two-handed grip support
    • Physics-driven interaction
    • Blueprint-only implementation
    • Modular and clean architecture

    📦 Climbing Movement Add-On

    Description Turn Your VR Interaction System into a Full Climbing Experience

    VRHPG Climbing Add-on extends VR Hands Procedural Grab Pro with a fully procedural, physics-driven climbing system designed specifically for VR.

    No canned animations. No fake movement. Just true procedural climbing.

    PLAYABLE DEMO

    FAB PAGE

    ⚠️ VR Hands Procedural Grab Pro Plugin is required for this framework!

    ⚡Quick Start Guide

    • Add asset to your project

    • Add ClimbingMovement component to your VR Pawn

    • Add tag CanClimb to any climbable actor
    • Change movement for all components inside climbable actor from Movable to Static
